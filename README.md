# CountryVote

In order to run this app, you need docker compose, add build flag if its your first time running the container:
```
docker compose up --build
```

Once the container is running, you can open your browser and go to `http://localhost:5173/` to access the app.

For more information about the fronend you can take a look at its [readme](frontend/README.md)

For more information about the backend you can take a look at its [readme](backend/README.md)
